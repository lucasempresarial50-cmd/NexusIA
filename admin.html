Configurações do Sistema
🔑 Variáveis de Ambiente (Vercel)

Configure estas variáveis em Vercel → Settings → Environment Variables:

GROQ_API_KEY
Sua chave da Groq (gsk_...)
JWT_SECRET
String aleatória longa para assinar tokens. Ex: nexus-prod-2025-super-secret-xyz
ADMIN_EMAIL
Seu e-mail de administrador
ADMIN_PASS
Sua senha de administrador (use algo forte!)
USERS_JSON
JSON com array de usuários (gerado automaticamente ao criar clientes)
Como adicionar um cliente
1
Clique em Adicionar Cliente no menu
2
Preencha nome, e-mail, senha inicial e selecione o plano
3
Clique em Gerar acesso — será gerado um JSON com os dados do usuário
4
Cole o JSON em Vercel → Settings → Env → USERS_JSON
5
Faça Redeploy no Vercel para aplicar
6
Envie ao cliente: link do sistema + e-mail + senha
